# 课堂实录标注与分析系统

## 项目简介

基于生成式人工智能的课堂实录视频标注与分析 Agent，专攻教学视频的智能标注、互动数据量化、教学行为评估与改进建议生成，帮助教师实现数据驱动的深度教学反思。

## 核心功能

### 1. 视频上传与管理
- 支持 MP4、AVI、MOV、WebM 等主流视频格式
- 拖拽上传与文件选择两种方式
- 课程基本信息录入（学科、年级、课型、课题）

### 2. 智能标注面板
- **快捷标注**：播放视频时一键添加教学行为标签
- **手动标注**：自定义标注类型、时间范围和备注
- **AI自动标注**：基于教学行为识别模型自动生成标注（需后端服务）
- **标注分类**：教师行为、学生行为、互动、教学法、媒体使用等
- **标注导出**：支持 CSV/JSON 格式导出

### 3. 时间轴分析
- 按分钟聚合的教学行为堆叠柱状图
- 可视化时间轨道，展示各类教学行为的分布
- 教学行为占比实时统计
- 多视图切换（完整/教师/学生/互动）

### 4. 多维数据分析
- **弗兰德斯互动分析 (FIAS)**：教师言语比率、学生言语比率、沉默比率
- **S-T 教学分析**：Rt(教师行为占有率)、Ch(行为转换率)、教学模式判定
- **提问类型分布**：记忆性、理解性、应用性、分析性、评价性、创造性
- **课堂节奏分析**：活动密度曲线与课堂节奏指标
- **互动热度分布**：各时间段教学行为堆叠分布
- **教学环节时间分配**：各教学环节占比变化趋势

### 5. 反思报告
- 自动生成结构化教学反思报告
- 基于理论框架的客观洞察
- 量化数据驱动的改进建议
- 支持导出和打印

## 技术栈

- **前端**：原生 JavaScript (ES Modules) + Chart.js + HTML5/CSS3
- **构建工具**：Vite
- **后端**：Node.js + Express
- **视频处理**：HTML5 Video API
- **数据可视化**：Chart.js 4.x

## 理论框架

### 弗兰德斯互动分析系统 (FIAS)
将课堂言语行为分为教师言语、学生言语、沉默/混乱三大类，通过量化分析评估课堂互动结构。

### S-T 教学分析法
通过 Rt（教师行为占有率）和 Ch（行为转换率）两个指标，将教学模式分为讲授型、练习型、对话型、混合型四种。

## 使用流程

1. **上传视频** → 选择课堂实录视频，填写课程基本信息
2. **智能标注** → 播放视频，使用快捷标注或手动标注教学行为
3. **时间轴分析** → 查看教学行为的时间分布
4. **数据分析** → 查看 FIAS、S-T 等多维分析图表
5. **反思报告** → 生成并导出教学反思报告
